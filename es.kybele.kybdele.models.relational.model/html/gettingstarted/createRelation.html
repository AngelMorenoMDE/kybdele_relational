Create Relation

To create a relation we know that there are two ways, the first being by clicking on the Relation Tool GMF tool palette.




Below, we click where you want to place the Relation. The relation is automatically created and the name will focus, for this to be changed.




After entering the name of the relation will have been created correctly.





the second way is by placing the cursor on the diagram. One dialog where we can see the Relation Tool appears.




Select it and the relation appear in that position, define the name and relation will have been created successfully
