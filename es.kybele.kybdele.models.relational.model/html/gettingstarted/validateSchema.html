Validate schema

To validate the schema must first know the 3 levels of severity:

- Info > Generally useful information. Info I want to always have available but usually don't care about under normal circumstances.
- Warning > Minor user-level exceptions that are not expected during normal processing.
- Error > Error is used to log all unhandled exceptions diagram.


Validate schema via Edit → Validate.




Below, open Problems View via Windows → Show View → Other... → Problems In this view you can see all the validations.




If you double click on error or warning that refers to a single element, this element is selected automatically in the diagram
