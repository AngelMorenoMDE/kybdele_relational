Add Attribute

To add an attribute to a relation we know that there are two ways, the first being by clicking on the Attribute Tool GMF tool palette.




Below, click inside the relationship to which you want to add the attribute




The attribute is automatically created and the name will focus, for this to be changed.




the second way is by placing the cursor between the two parenthesis of the relation to which we add an attribute. One dialog where we can see the Attribute Tool appears.




After entering the name of the attribute will have been created correctly.
